### 🌐 Vimium Shortcuts
Master browser navigation without touching the mouse.

Some favorites:

| Shortcut | Action |
|---------|--------|
| `j` | Scroll down |
| `k` | Scroll up |
| `J` | Next tab |
| `K` | Previous tab |
| `f` | Open links with hints |
| `H` | Go back |
| `L` | Go forward |
